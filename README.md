# Agatha Christie: Death on The Cards

Juego de cartas full-stack inspirado en mecánicas de deducción y resolución de misterios.

El proyecto está dividido en frontend y backend, desarrollados como componentes independientes.

## Estructura del repositorio

- `frontend/`: aplicación cliente
- `backend/`: API y lógica del juego

Cada componente cuenta con su propio README con instrucciones de instalación y ejecución.

## Ejecución en desarrollo

Para ejecutar el sistema, seguir las instrucciones específicas de cada componente:

- Frontend: `frontend/README.md`
- Backend: `backend/README.md`
