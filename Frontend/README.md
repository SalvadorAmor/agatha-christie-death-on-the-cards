# Frontend
## Aplicacion creada con React + TypeScript + Vite

## Dependencias
- Node.js
- npm

## Tecnologias
- [React] --> Libreria para la UI
- [Typescript] --> Tipado estatico
- [Vite] --> Servidor de desarrollo 
- [Vitest] --> Framework para testear 

## Para instalar: 
Clonar el repositorio y luego correr
- ``npm install``

## Para inciar el proyecto
- ``npm run dev``
- Esto levanta la aplicacion en ``http:localhost:5173``
